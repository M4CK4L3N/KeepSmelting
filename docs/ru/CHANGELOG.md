# Список изменений

## v0.2.0

### Новые возможности
- **Симуляция цепочки с воронками** — догонялка теперь симулирует полный конвейер: бочка→воронка→печь→воронка→бочка, с учётом пропускной способности и направления воронок
- **Унифицированные команды** — `simulate` и `test` работают с любыми печами, не только Iron Furnaces
- **Поддержка ванильных печей** — догонялка и тесты для обычных печей, коптилен и плавильных печей

### Улучшения
- **Более точная догонялка** — учёт ресурсов во всех контейнерах (печь + воронки + бочки), выталкивание результата через всю цепочку
- **Улучшенные тесты** — ванильные тестовые схемы расставляют воронки в реалистичных конфигурациях
